Interview Scheduler

Use this app to determine an optimal scheduling of interviews given several possible anounced dates.

Instructions

  • Put each program + possible interview dates on a new line.
  • Separate dates from program name with ' - '
  • Separate dates with ', '
  • Dates can be mm/dd/yy, or even abbrev_day mm/dd/yy (eg: 9/27/14 or Sat 9/27/14)
  • Program name appended with asterisk (eg Prog3*) will display in green to signify the interview has been actually scheduled
  • Example
    Program1 - Sat 12/6/14, Sat 12/13/14
    Program2 - Sat 12/13/14, Sat 12/27/14
    Program3* - 12/20/14

Source

Code available on GitHub.
The scheduler utilizes the Ford-Fulkerson algorithm.
