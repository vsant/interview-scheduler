Interview Scheduler

Use this app to determine an optimal scheduling of interviews given several possible anounced dates.

Instructions

  • Put each program + possible interview dates on a new line.
  • Separate dates from program name with ' - ' (spaces matter)
  • Separate dates from each other with ', ' (spaces matter)
  • Each date should be of format abbreviated_day mm/dd/yy (eg. Sat 9/27/14)
  • Example
    Program1 - Sat 12/6/14, Sat 12/13/14
    Program2 - Fri 12/12/14, Sat 1/10/15
